# Sentinelle — IA Vivante
**Créée par Nicolas Breidi / L'Aube Étoilée Technologies**

## Lancer
```bash
pip install ollama
ollama serve &
python main.py
```

## Structure
- cerveau/ — LLM Ollama
- voix/    — Whisper + TTS
- vision/  — Reconnaissance faciale
- taches/  — Exécution de commandes
- memoire/ — Historique conversations
